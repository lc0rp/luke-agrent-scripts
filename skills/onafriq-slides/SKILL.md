# Onafriq Slides

Use this skill for any Onafriq slide work.

## Source files

- Slide master templates (pptx): `references/onafriq-slides-template.pptx`
  - Slide master templates (pdf): `references/onafriq-slides-template.pdf` (can use for reference)
  - Slide master templates (montage): `references/onafriq-slides-template-montage.png` (can use for reference)
- Brand guide: `references/onafriq-brand-guide-v4-web.pdf`
- Slide-focused brand summary: `references/onafriq-slide-brand-rules.md`

Read `references/onafriq-slide-brand-rules.md` before making slide changes. Go back to the PDF only if you need deeper brand context.

## Core rule

Treat the PowerPoint template as the production design system. Build from it, not around it.

## Required workflow

Follow this sequence every time:

1. Duplicate the master templates `references/onafriq-slides-template.pptx`.
2. Work only in the duplicated file. Use only the master templates from this file.
3. Review the existing slides in that copy and find the slide whose structure best matches the new content.
4. Duplicate that source slide.
5. Replace the placeholder or existing content on the duplicated slide.
6. Preserve the original layout architecture, type hierarchy, footer treatment, and brand styling.
7. Repeat by duplicating more template slides as needed.
8. Remove unused template/reference slides before final delivery.

## Non-negotiable rules

- Never start from a blank deck.
- Never edit the original reference slides directly.
- Never invent a fresh layout when an existing template slide can be adapted.
- Never remove or casually redesign footer elements that already contain the logo and page number treatment.
- Never swap out fonts, colors, or spacing rules for something that feels merely "close enough."
- Never redraw, restyle, distort, rotate, outline, or recolor the logo outside approved treatments.
- Never force dense copy into a slide by shrinking everything until the house style breaks.

## Layout discipline

- Use the closest matching template slide first, even if it requires light copy adaptation.
- Preserve the placement logic of titles, subtitles, body copy, images, tables, charts, and dividers.
- Keep anchors, margins, padding, and alignment consistent with the chosen template slide.
- If content does not fit, cut copy, split content across slides, or choose a better source slide.
- If no template slide supports the content cleanly, make the least-invasive adaptation of an existing layout and call that out.

## Brand discipline

- Match the Onafriq tone of voice in slide copy: accessible, knowledgeable, inspiring, resonant, and confident.
- Keep language simple, clear, and professional.
- Preserve the distinctive Onafriq look: restrained, polished, high-clarity, and recognizably corporate rather than decorative.
- Favor the exact fonts, colors, and graphic treatments already present in the template.
- Use the brand-summary reference to sanity-check typography, color usage, logo handling, imagery, and pattern usage.

## Copy discipline

- Be practical and pragmatic.
- Do not use em dashes. Use a semicolon or rewrite the sentence.
- Never state the obvious unless asked, unless it is required for the reasoning, or unless the audience is unlikely to know it.
- Keep labels and section titles lean. If a list is examples, call it `examples`, not `Examples (summarized)`.
- Avoid common LLM writing fingerprints. Keep phrasing direct, natural, and low-friction.
- Do not use contrastive repetition patterns such as `not X, but Y`, `This does X, not just Y`, `The goal is not X. The goal is Y.`, or `This is not about X. It is about Y.` State the intended outcome in one clean sentence.
- Do not use overworked slogan patterns such as `X is the unlock`.

## Content fitting rules

- Rewrite for brevity before resizing text.
- Split dense ideas across multiple slides before breaking the layout.
- Prefer a clean, readable deck over squeezing in every source detail.
- Keep visual emphasis aligned with the template's existing headline and body hierarchy.

## Final quality bar

Before handing off the deck, confirm:

- every kept slide came from a duplicated template slide
- unused reference slides were removed
- footer, logo, and page numbering remain intact
- fonts and colors still match the template and brand guide
- the deck looks like native Onafriq material rather than adapted third-party slides
